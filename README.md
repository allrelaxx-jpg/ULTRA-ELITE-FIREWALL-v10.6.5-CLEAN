# ULTRA ELITE FIREWALL v10.6.5 CLEAN

Minimal production firewall (no monitoring, no extra services).

## Install

curl -sL "https://raw.githubusercontent.com/allrelaxx-jpg/ULTRA-ELITE-FIREWALL-v10.6.5-CLEAN/main/install.sh?nocache=$(date +%s)" | bash

## Features

- nftables firewall
- NAT (VPN ready)
- Cloudflare protection (80/443)
- GeoIP access control
- SSH brute protection

## Ports

- 80/443 → Cloudflare only
- 22 → Geo restricted
- 8443 → Reality
- 51820 → WireGuard
- 32690–32700 → AmneziaVPN
- 3000 → Dashboard (VPN only)
- 4000 → CYBER AI CORE
